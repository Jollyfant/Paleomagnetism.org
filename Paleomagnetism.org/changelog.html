    Version 1.1.0 2016-08-31

    A new version has been released with many new features and core changes to the codebase. Please report any inconsitencies or bugs. In particular, previously created .dir files will be patched when they are loaded to the application. To get the patched data files, re-download the data after it has been loaded to the application. Please keep a backup of old .dir files in any case.

    • Fixed bug in VGP > DIR calculation [see #9] for PalPole
    • Added magnetrostratigraphic module (under misc. portal)
    • Added plate boundaries to map module
    • Added option to view verified data by Paleomagnetism.org
    • Added option to group interpretations
    • Added more ages options for sites
    • Added options to sort specimens in interpretation portal
    • Added tickmarks to NTR plots
    • Added option to add Euler poles to map module
    • Improved importing of files
    • Support for offline exporting of .png and .svg
