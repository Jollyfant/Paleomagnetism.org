    Introduction

    Paleomagnetism.org is designed for the interpretation and statistical evaluation of Paleomagnetic data. The website promotes the ability to share Paleomagnetic data between researchers in an online environment.

    The website is divided in to three portals described below:

    1. Statistics Portal

    The statistical portal includes ways to visualize and evaluate paleomagnetic data (declination - inclination pairs) using common procedures.

    2. Interpretation Portal

    The interpretation portal allows for the analysis of demagnetization data and interpretation using eigenvector analysis (Kirschvink, 1981).

    3. Miscellaneous Portal

    This portal includes a collection of tools with a distinctive input and output. Currently two tools are available:

    • A Bootstrapped Oroclinal Test
    • Net Tectonic Rotation Analysis
