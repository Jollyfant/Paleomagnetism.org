    Directions vs Stratigraphic Level

    This modules illustrates inclination and declination against an arbitrary stratigraphic level. A user-specified stratigraphic polarity scale can be added by clicking the Import Stratigraphy

    The input format for this file is (lower, upper) level for filled blocks specified on each line. An example format of this file can be downloaded here. Scaling for the stratigraphic column is done automatically to match the axes of the inclination/declination plots.
